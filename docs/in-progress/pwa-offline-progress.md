1. **Create `src/lib/utils/api.ts`** — A fetch wrapper (`apiFetch`) that:
   - For write methods (POST/PATCH/DELETE): if `navigator.onLine` is false, calls `enqueue()` from `offline-queue.ts` and returns a synthetic `Response` with `{ queued: true }`
   - For read methods or when online: passes through to normal `fetch()`

   - Export `startSyncListener()` function
