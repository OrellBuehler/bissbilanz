    2. Create WeightLogForm.svelte — inline form (not modal) with:
       - Number input for weight in kg (step="0.1", min="20", max="500")
       - Date input defaulting to today (user-local date)
       - Optional notes text input
       - Submit button. On submit, POST to /api/weight with { weightKg, entryDate, notes }
       - Dispatch a custom event or call an `onLogged` callback prop so parent can refresh data
       - Use shadcn-svelte Input, Button components. Show validation errors inline.

    3. Create WeightHistoryList.svelte — receives entries array as prop:
       - Display entries in a list/table: date, weight (kg, 1 decimal), notes, time logged
       - Each entry has edit and delete actions (icon buttons)
       - Edit: inline editing — click edit to make weight/notes fields editable, save with PATCH /api/weight/[id]
       - Delete: confirm dialog, then DELETE /api/weight/[id]
       - Call `onChanged` callback prop after edit/delete so parent can refresh
       - Order by loggedAt DESC (most recent first)
       - Use shadcn-svelte Card or simple styled list matching existing app patterns

    4. Create WeightChart.svelte — receives chart data (with movingAvg) and selected range as props:
       - Use layerchart LineChart with two series:
         - Raw weight (primary color, e.g., purple #8B5CF6, with data points)
         - 7-day moving average (orange #F97316, smooth line, no points)
       - Follow CalorieTrendChart pattern for axis formatting, tooltips, responsive sizing
       - X-axis: dates. Y-axis: weight in kg. Tooltip shows both values.
       - Include ChartRangeSelector with ranges [7d, 30d, 90d, all]
       - Dispatch range change event or call `onRangeChange` callback

    5. Add i18n messages to en.json and de.json for all weight-related strings:
       - nav_weight, weight_page_title, weight_log, weight_history, weight_chart
       - weight_kg_label, weight_date_label, weight_notes_label, weight_save
       - weight_actual, weight_trend, weight_edit, weight_delete, weight_confirm_delete
       - weight_no_entries, weight_range_7d, weight_range_30d, weight_range_90d, weight_range_all
       - weight_latest (for dashboard widget)
